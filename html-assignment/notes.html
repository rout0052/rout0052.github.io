Homepage Notes
<a>
The a tag, or anchor tag, is how links are made in HTML. This could be external links to other websites, links to other pages on the same site, or even a link to jump to a specific section of the page. The attribute 'href' is where the URL is added.
<hr>
hr stands for horizontal, and adds a horizontal line across the screen. This should only be used for a thematic break in content, both for best accessibility practices (screen readers picking up the break in content, versus skipping a decorative split) and because you have more control of CSS elements.
<nav>
The nav tag, or navigation tag, is a parent tag that holds a group of navigation links. Note: this should only be used when showing multiple navigation links, as single links can simply use the anchor tag.
<img>
The img tag, or image tag, embeds an image into an HTML page. The 'src' attribute has the link or file path of the image file.
<h1>
There are six heading tags, ranging in size from h1-h6. h1 is the largest of these, and should be used for the primary heading on the page, or the title.
